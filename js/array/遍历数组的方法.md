- for, for...in, for...of

for...in 语句以任意顺序迭代一个对象的除 Symbol 以外的可枚举属性，包括继承的可枚举属性

- forEach
- filter, map
- every, some
- reduce, reduceRight
- find, findIndex
- keys, values, entries

Object.keys() 静态方法返回一个由给定对象自身的可枚举的字符串键属性名组成的数组。
