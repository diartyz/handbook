## in operator

如果指定的属性在指定的对象或其原型链中，则 in 运算符返回 true。

```javascript
propertyKey in target
```

## Reflect.has()

和 in 运算符 的功能完全相同

```javascript
Reflect.has(target, propertyKey)
```

## Object.prototype.hasOwnProperty()

```javascript
hasOwnProperty(prop)
```

## Object.hasOwn()

Object.hasOwn() 旨在取代 Object.prototype.hasOwnProperty()

```javascript
Object.hasOwn(obj, prop)
```
